## 🚀 How to Run

### Root Portfolio
```bash
npm install
npm run dev
```
Runs on http://localhost:5173

### Individual Apps
```bash
# Tip Calculator
cd apps/tip-calculator
npm install
npm run dev  # Local development
npm run deploy  # Deploy to GitHub Pages

# FAQ Accordion
cd apps/faq-accordion
npm install
npm run dev  # Local development
npm run deploy  # Deploy to GitHub Pages
```
